# Machine Learning Engineer Interview Questions

---

**Q: What is the difference between bagging and boosting?**

**A:** Bagging trains multiple models in parallel on random subsets; boosting trains models sequentially, each focusing on previous errors.

---

**Q: How do you deploy a machine learning model to production?**

**A:** Use APIs (Flask, FastAPI), Docker, CI/CD, cloud services (AWS Sagemaker, GCP AI Platform), and monitor performance.

---

**Q: What is feature selection and why is it important?**

**A:** Feature selection chooses relevant features to improve model performance and reduce overfitting.

---

**Q: How do you handle imbalanced datasets?**

**A:** Use resampling (oversample minority, undersample majority), class weights, or specialized algorithms (SMOTE, XGBoost).

---

**Q: What is model drift and how do you detect it?**

**A:** Model drift is performance degradation over time due to data changes. Detect using monitoring, statistical tests, and retrain as needed.
