# Data Scientist Interview Questions

---

**Q: Explain the difference between supervised and unsupervised learning.**

**A:** Supervised learning uses labeled data; unsupervised learning finds patterns in unlabeled data.

---

**Q: What is regularization? Why is it important?**

**A:** Regularization (L1/L2) penalizes large weights to prevent overfitting and improve generalization.

---

**Q: How do you handle missing data?**

**A:** Impute (mean/median/mode), use models that handle missing values, or drop rows/columns.

---

**Q: What is the bias-variance tradeoff?**

**A:** Bias is error from wrong assumptions; variance is error from sensitivity to small fluctuations. The tradeoff is balancing underfitting and overfitting.

---

**Q: How do you evaluate a classification model?**

**A:** Use metrics like accuracy, precision, recall, F1-score, ROC-AUC, confusion matrix.
