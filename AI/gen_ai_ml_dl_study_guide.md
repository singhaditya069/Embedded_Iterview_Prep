# Study Guide: Generative AI, Machine Learning, and Deep Learning

---

## 1. Generative AI
- What is Generative AI? Examples: ChatGPT, DALL-E, Midjourney.
- Key concepts: Generative models, GANs, VAEs, diffusion models, prompt engineering.
- Applications: Text generation, image synthesis, code generation, music, art.
- Study topics: Transformer architecture, attention mechanism, large language models (LLMs), prompt design, fine-tuning, RLHF.

## 2. Machine Learning (ML)
- Types: Supervised, unsupervised, semi-supervised, reinforcement learning.
- Algorithms: Linear/logistic regression, SVM, decision trees, random forest, k-NN, clustering (k-means, DBSCAN), PCA.
- Model evaluation: Cross-validation, confusion matrix, ROC-AUC, precision/recall, F1-score.
- Feature engineering: Scaling, encoding, selection, extraction.
- Overfitting/underfitting, regularization (L1/L2), bias-variance tradeoff.

## 3. Deep Learning (DL)
- Neural networks: Perceptron, MLP, activation functions (ReLU, sigmoid, tanh).
- CNNs: Convolution, pooling, filters, architectures (LeNet, AlexNet, ResNet).
- RNNs: Sequence modeling, LSTM, GRU, attention.
- Training: Backpropagation, optimizers (SGD, Adam), loss functions, batch normalization, dropout.
- Frameworks: TensorFlow, PyTorch, Keras.

## 4. Study Resources
- Books: "Deep Learning" (Goodfellow), "Hands-On ML with Scikit-Learn, Keras, and TensorFlow" (Aurélien Géron), "Pattern Recognition and Machine Learning" (Bishop).
- Courses: Coursera (Andrew Ng ML/DL), fast.ai, DeepLearning.AI, Stanford CS231n/CS224n.
- Practice: Kaggle, LeetCode (ML), Papers With Code.

## 5. Interview Preparation
- Review ML/DL algorithms, math (linear algebra, probability, calculus), coding (Python, NumPy, pandas, scikit-learn, PyTorch/TensorFlow).
- Practice case studies, system design for ML, and explainability/ethics questions.
