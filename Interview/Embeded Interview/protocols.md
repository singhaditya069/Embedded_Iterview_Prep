# Communication Protocols Interview Questions

## Concepts
- Differentiate between UART, SPI, I2C, CAN, USB, Bluetooth, Ethernet.
- Explain packet structure and framing for a selected protocol.
- What are CRC and checksum? Why are they used?

## Theoretical
- Discuss error detection vs correction.
- Explain flow control and handshaking mechanisms.

## Coding
- Write code to initialize an I2C master and perform a read transaction.
- Show how to parse an incoming UART frame with start/stop bits and parity.

## MAANG-Level
- Devise a custom lightweight protocol for low-power mesh networks.
- Evaluate latency and throughput requirements for a mixed protocol system.
