# Device Driver Interview Questions

## Concepts
- Explain the role of a device driver in embedded systems.
- What is a character driver vs a block driver?
- Describe memory-mapped I/O and ports.
- How does interrupt handling work? Write ISR constraints.
- Explain DMA and its use in drivers.

## Theoretical
- Discuss driver layering and hardware abstraction.
- How do you manage concurrency and reentrancy in drivers?
- Explain driver initialization sequence and power management.

## Coding
- Write a simple UART driver initialization routine for an ARM microcontroller.
- Implement an SPI transfer using polling, interrupts, and DMA.
- Show pseudo-code for registering an interrupt and corresponding ISR.

## MAANG-Level
- Design a driver framework supporting hot-pluggable modules.
- Analyze trade-offs between synchronous and asynchronous driver APIs.
