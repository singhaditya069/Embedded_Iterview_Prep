# Basics Interview Questions (Hardware Product Companies)

---

**Q: What is the difference between microprocessor and microcontroller?**

**A:** Microprocessor is CPU only, needs external peripherals; microcontroller integrates CPU, memory, and peripherals on a single chip.

---

**Q: What is endianness? Difference between little-endian and big-endian?**

**A:** Endianness is the order of bytes in multi-byte data. Little-endian: least significant byte first. Big-endian: most significant byte first.

---

**Q: What is polling? How is it different from interrupt?**

**A:** Polling is actively checking device status in a loop; interrupt is hardware-driven, CPU is notified only when needed.

---

**Q: What is a pull-up and pull-down resistor?**

**A:** Pull-up connects input to Vcc, pull-down to GND, ensuring a defined logic level when no active driver is present.

---

**Q: What is metastability in digital circuits?**

**A:** Metastability occurs when a flip-flop receives input changes near the clock edge, causing unpredictable output. Use synchronizers to mitigate.

---

**Q: What is the difference between static and dynamic RAM?**

**A:** SRAM uses flip-flops, is faster and more expensive; DRAM uses capacitors, is slower and cheaper, needs refresh.

---

**Q: What is setup and hold time?**

**A:** Setup time: minimum time data must be stable before clock edge. Hold time: minimum time data must remain stable after clock edge.
